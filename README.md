# A Custom research agent

My simple research assistant agent.

## To clone & run this project

`` git clone https://github.com/murigugitonga/custom_research_assistant.git
`` python -m venv\venv
`` pip install -r requirements.txt
`` get your API key from your LLM provider of choice(OpenAI or anthropic)

Feel free to tweak & add functionality according to your preference.
